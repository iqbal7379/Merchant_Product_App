# Merchant Product App

A **Java application** using **Hibernate ORM** to manage merchants and their products efficiently. 
This application allows merchants to add, update, and view products, while users can browse and search products easily.

---

## Features

- **Merchant Management**: Add, edit, and remove merchants.
- **Product Management**: Add, update, delete, and view products for each merchant.
- **Search Products**: Search products by name, category, or merchant.
- **Database Integration**: Uses **Hibernate ORM** with MySQL/PostgreSQL for persistent data storage.
- **Exception Handling**: Handles database and application errors gracefully.

---

## Tech Stack

- **Programming Language**: Java
- **ORM**: Hibernate
- **Database**: MySQL / PostgreSQL
- **Build Tool**: Maven
- **Version Control**: Git & GitHub
